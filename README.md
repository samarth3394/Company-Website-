## Project Structure
The project follows a standard, professional web architecture:
```text
BhartiNexusTech/
├── assets/
│   ├── css/       # Stylesheets (index.css, about.css, etc.)
│   ├── images/    # Company logos and team portraits
│   ├── js/        # Interactive scripts and animations
│   └── videos/    # Background hero videos
├── index.html     # Homepage
├── about.html     # About Us & Leadership page
├── privacy.html   # Privacy Policy
└── terms.html     # Terms of Service
```

## Technologies Used
*   **HTML5** for semantic structuring.
*   **Vanilla CSS3** for responsive layouts and animations (No external CSS frameworks required).
*   **Vanilla JavaScript** for interactive elements like mobile navigation and canvas-based particle animations.

## How to Run Locally
Since this is a static website, you do not need any complex build tools. 
1. Clone the repository.
2. Open `index.html` in any modern web browser to view the site.
3. For the best development experience, consider using a local development server like [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) in VS Code to see live changes.
