# Bharti Nexus Technologies

Welcome to the official website repository for **Bharti Nexus Technologies**, an AI-driven, futuristic enterprise technology company.

## 🚀 Overview

This repository contains the source code for the Bharti Nexus official landing page. The website features a modern, clean, and highly responsive single-page architecture built to showcase the company's core capabilities in Artificial Intelligence, Machine Learning, and Enterprise Automation.

### ✨ Key Features

- **Premium UI/UX:** Clean, monochromatic black-and-white theme with dark glassmorphism effects.
- **Dark/Light Mode:** Seamless toggle between light and dark themes using CSS variables.
- **Interactive Animations:** Scroll reveals, hover effects, and custom CSS keyframe animations.
- **Responsive Layout:** Fully optimized for desktops, tablets, and mobile devices.
- **Glassmorphic Navigation:** A fixed, blur-backdrop navigation bar for a seamless browsing experience.

## 📁 Project Structure

- `index.html` - The core HTML file containing all markup, inline CSS styling, and JavaScript logic.
- `logo2.png` - Primary company logo.
- `founder.png`, `cofounder.png`, `rohan.png` - Leadership profile images.

## 🛠️ Technologies Used

- **HTML5:** Semantic structuring.
- **CSS3:** Flexbox, Grid, custom properties (variables), backdrop filters, and smooth transitions.
- **JavaScript (Vanilla):** DOM manipulation, scroll event handling, theme toggling, and intersection observers for scroll animations.

## 💻 How to Run Locally

Since this is a static website, no complex backend setup is required.

1. Clone the repository:
   ```bash
   git clone https://github.com/samarth3394/Company-Website-.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Company-Website-
   ```
3. Open `index.html` directly in any modern web browser (Chrome, Firefox, Safari, Edge).
   - *Alternatively, use a tool like VS Code "Live Server" extension for a better development experience.*
